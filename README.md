# TableCRM Test Project

Это тестовый проект для работы с системой TableCRM, реализованный на **Next.js (App Router)** и **React/TypeScript**.  

---

## Стек технологий

- **Next.js 13+ (App Router)**
- **React 18+**
- **TypeScript**
- **Tailwind CSS**
- **OpenAI API** (для генерации описаний через нейросеть)
- **TableCRM API** (для работы с товарами и категориями)

---

## Основной функционал

### 1. Управление товарами

- Создание карточки товара с обязательными полями:
  - Название (`name`)
  - Краткое описание (`shortDescription`)
  - Длинное описание (`longDescription`)
  - Артикул (`code`)
  - Цена (`marketplace_price`)
  - Категория (`category`)
	- Ключевые слова `seo_keywords`
  - TODO: SEO-параметры (`seo_title`, `seo_description`)
  - TODO: Геолокация (`address`, `latitude`, `longitude`)
  - TODO: Тип cashback (`cashback_type`)
- TODO: Проверка уникальности артикула перед добавлением (как на клиенте, так и на сервере).

### 2. Генерация описаний через нейросеть

- Пользователь вводит название товара.
- Кнопка **“Сгенерировать описания”** создаёт короткое и длинное описание через OpenAI API.
- Если генерация не удалась, возвращается fallback с `"Описание недоступно"` и `"Длинное описание недоступно"`.
- Кнопка **неактивна**, если поле `Название` пустое.

### 3. SEO-параметры

- Возможность добавления ключевых слов (`seo_keywords`) через tag-input.  
- SEO Title и SEO Description можно заполнять вручную или через генерацию (по желанию, если подключить OpenAI).

### 4. UI/UX

- Используется Tailwind для стилизации.
